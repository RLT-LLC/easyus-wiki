# О заказах

**Заказы** - часть easyus, предназначенная для работы со всеми входящими заказами. Менеджеры могут добавлять заказы вручную, а также работать с автоматически добавленными через интеграции (например, заявки с сайта) 

У каждого заказа есть
- **Ответственные** - сотрудники, которые контролируют выполнение
- **Исполнители** - сотрудники, которые выполняют заказ.

Исполнителей, как и ответственных на задаче может быть неограниченное количество.

**В этой статье Вы научитесь:**

-   Создавать и изменять заказы
-   Пользоваться чатом и другими функциями заказов
-   Обрабатывать заявки с сайта
-   Создавать и изменять колонки заказов
-   Использовать поиск и фильтр по заказам

*Подключение сайта к easyus рассматривается в статье "Интеграции"*

# Создание нового заказа

> Доступно функционалям: **Менеджер, Менеджер-распределитель, Модератор**
{.is-info}



###### На версии для ПК

1.  Перейти во вкладку “Заказы”
2.  Нажать на кнопку “Новый заказ”
3.  Заполнить поля информации о заказе
4.  Для сохранения нажать на кнопку "Создать"  
     

###### На мобильной версии

1.  Перейти во вкладку “Заказы”
2.  Нажать на “+” в правом нижнем углу
3.  Заполнить поля информации о заказе
4.  Для сохранения нажать на галочку

# Редактирование заказа

> Доступно функционалям: **Менеджер, Менеджер-распределитель, Модератор**
{.is-info}

###### На версии для ПК

1.  Перейти во вкладку “Заказы”
2.  Нажать на карточку заказа, который подлежит изменению
3.  Нажать на кнопку “Редактировать”
4.  Изменить поля
5.  Нажать на кнопку “Сохранить”  
     

###### На мобильной версии

1.  Перейти во вкладку “Заказы”
2.  Нажать на карточку заказа, который подлежит изменению
3.  Нажать на иконку “Карандаш” возле названия заказа
4.  Изменить поля
5.  Для сохранения нажать на галочку

# Завершение заказа

После того как заказ выполнен, необходимо перенести его в архив. Заказ можно пометить выполненным или проваленным. Позже все заказы можно будет просмотреть в архиве, на главной странице.

> Доступно функционалям: **Менеджер, Менеджер-распределитель, Модератор**
{.is-info}

###### На версии для ПК

1.  Перейти во вкладку “Заказы”
2.  Зажать левой кнопкой мыши по заказу, который необходимо завершить
3.  В правом нижнем углу появятся две колонки "Реализовано" и "Не реализовано"
4.  Перенесите заказ в нужную колонку
     

###### На мобильной версии

1.  Перейти во вкладку “Заказы”
2.  Нажать на карточку нужного заказа
3.  Нажать на иконку “Три точки” в правом верхнем углу
4.  Нажать на "Завершить заказ" или "Провалить заказ"

# Чат в заказе
Чат в заказе может использоваться для самых разнообразных целей. Например: для хранения информации и файлов о заказе, для общения между сотрудниками и т.д.

> Доступно функционалям: **Исполнитель, Менеджер, Менеджер-распределитель, Модератор**. Для использования чата необходимо быть назначеным исполнителем или ответственным на заказе
{.is-info}


###### На версии для ПК

1. Перейти во вкладку "Заказы"
2. Нажать на карточку интересующего заказа
3. В правой части экрана находится чат, если вы участник заказа, то можете начать переписку
3.1 Для прикрепления файла необходимо нажать на иконку "Скрепка"
3.2 Для отправки сообщения необходимо нажать Enter или нажать на иконку "Самолет"

###### На мобильной версии

1. Перейти во вкладку "Заказы"
2. Нажать на карточку интересующего заказа
3. Нажать на иконку "Самолет" в правом нижнем углу
3.1 Для прикрепления файла необходимо нажать на иконку "Скрепка"
3.2 Для отправки сообщения необходимо нажать Enter или нажать на иконку "Самолет"


# Приватный чат в заказе
Приватный чат предназначен для хранения заметок и обсуждения деталей заказа между менеджерами. **Приватный чат видят только ответственные лица заказа, исполнителям он не доступен**

> Доступно функционалям: **Менеджер, Менеджер-распределитель, Модератор**. Для использования приватного чата необходимо быть назначеным исполнителем или ответственным на заказе
{.is-info}



###### На версии для ПК

1. Перейти во вкладку "Заказы"
2. Нажать на карточку интересующего заказа
3. В чате в правом верхнем углу нажмите на икнонку "Замок"
4. Использование приватного чата аналогично использованию обычного чата

###### На мобильной версии

1. Перейти во вкладку "Заказы"
2. Нажать на карточку интересующего заказа
3. Нажать на иконку "Самолет" в правом нижнем углу
4. Переключатель "Приватный режим" вверху экрана перевести в активированный режим
5. Использование приватного чата аналогично использованию обычного чата

# Создание колонки заказов
> Доступно функционалям: **Модератор**. 
{.is-info}


###### На версии для ПК

1. Перейти во вкладку "Заказы"
2. Пролистать доску заказов в крайнее правое положение
3. Нажать на кнопку "Добавить новую колонку"
4. Ввести название колонки и выбрать её цвет
5. Нажать на кнопку "Создать список"

###### На мобильной версии

1. Перейти во вкладку "Заказы"
2. Нажать на иконку "Шестеренка" в верхнем правом углу экрана
3. Нажать на кнопку "+" в правом нижнем углу экрана
4. Ввести название колонки и выбрать её цвет
5. Нажать на кнопку "Сохранить"

# Изменение позиции колонки заказов
> Доступно функционалям: **Модератор**. 
{.is-info}

###### На версии для ПК

1. Перейти во вкладку "Заказы"
2. Зажать левой кнопкой мыши на колонке, позицию которой нужно изменить
3. Перенести колонку в желаемое положение

###### На мобильной версии

1. Перейти во вкладку "Заказы"
2. Нажать на иконку "Шестеренка" в верхнем правом углу экрана
3. При помощи стрелочек возле названий колонок изменять их положение


# Поиск и фильтр по заказам
> Доступно функционалям: **Исполнитель, Менеджер, Менеджер-распределитель, Модератор**. 
{.is-info}

###### На версии для ПК

1. Перейти во вкладку "Заказы"
2. Нажать на кнопку "Фильтры", выбрать интересующие фильтры, нажать "Применить"
3. Для поиска ввести значение в строку "Поиск", выбрать интересующий результат из выпадающего списка

###### На мобильной версии

1. Перейти во вкладку "Заказы"
2. Нажать на иконку "Фильтр", выбрать интересующие фильтры. Нажать "Сохранить"
3. Для поиска необходимо нажать на иконку "Лупа", ввести интересующее значение в строку поиска

# Распечатка заказа

> Доступно функционалям: **Менеджер, Менеджер-распределитель, Модератор**. 
{.is-info}

> Недоступно на мобильной версии 
{.is-danger}

###### На версии для ПК

1. Перейти во вкладку "Заказы"
2. Нажать на интересующий заказ
3. Нажать на кнопку "Печать", которая находится возле кнопки "Редактировать"
